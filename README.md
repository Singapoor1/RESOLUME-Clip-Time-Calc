# RESOLUME-Clip-Time-Calc
A simple utility for calculating the total duration of selected clips in Resolume Arena. 

This tool works by parsing XML data copied directly from Resolume and summing the clip durations automatically.

✨ Features

Calculates total duration of multiple selected clips

Works with native XML data copied from Resolume Arena

Fast and lightweight

No direct integration required (copy–paste workflow)

🚀 How to Use

Open Resolume Arena

Select the clips you want to calculate

Press Ctrl + C to copy them

Switch to the calculator app

Make sure your keyboard layout is set to English

Paste the copied XML data into the input field

Click the Calculate button

The program will display the total duration of all selected clips.

🛠 Requirements

Resolume Arena

OS: Windows (if applicable — уточни если нужно добавить macOS)

Clipboard access

📌 Notes

The application works only with XML data copied directly from Resolume.

If the XML field is empty or incorrectly formatted, calculation will fail.

English keyboard layout is required for correct input handling.
